## 1. 執行摘要

### 1.1 公司簡介

**Jikko AI** 是一家總部位於香港的AI企業級SaaS公司，專注於為中小企提供一站式業務管理解決方案。我們的使命是通過人工智能技術，賦能香港34萬中小企實現數碼轉型，將寶貴的人力資源從繁瑣的行政工作中解放出來，專注於業務增長和價值創造。

作為香港首個深度本地化的AI企業管理平台，Jikko AI整合了HR、財務、銷售、客服和營運五大核心職能，讓中小企能夠以極低的成本享有過往只有大型企業才能負擔的企業級系統。

### 1.2 市場機會

香港是全球最具活力的商業中心之一，擁有超過34萬間中小企，佔全港企業總數超過98%。然而，這些企業長期面對著一個核心痛點：缺乏合適、實惠且易用的業務管理工具。

傳統企業軟件巨頭如SAP、Oracle的解決方案定價高昂、部署複雜，僅適合大型企業；而小型工具又缺乏整合性，導致企業需要在多個系統之間切換，效率低下。

根據IDC和Gartner的研究報告，全球SaaS市場正以19%的年均增長率快速擴張，而香港及大中華區的企業數碼化需求正在急速增長。政府近年亦推出「數碼轉型支援先導計劃」，提供合共5億港元資助中小企進行數碼升級。

### 1.3 解決方案

Jikko AI提供一個統一、智能且本地化的雲端平台，五大部門模組無縫協作：

- **HR人力資源** — 員工檔案、強積金管理、智能請假審批、考勤追蹤、績效評核
- **Finance財務** — 發票管理、費用報銷、薪酬發放、庫存管理、財務報告
- **Sales銷售** — 銷售管道管理、潛在客戶管理、客戶關係維護、報價生成、營銷自動化
- **Customer Service客服** — 智能工單系統、知識庫建設、AI客服機器人、SLA管理
- **Operations營運** — 項目管理、工作流程自動化、審批流程、模板庫

平台核心是建基於GPT-4的AI助手，能夠理解粵語和普通話，為企業提供24/7的智能顧問服務。

## 2. 問題與痛點

### 2.1 中小企面臨的系統困境

香港中小企在業務管理軟件的選擇上長期面對「高不成、低不就」的困境：

**選錯貴的代價：**
- SAP、Oracle等企業級系統的實施費用通常由HK$500,000起跳
- 每年授權費高達HK$100,000至HK$1,000,000
- 部署周期長達6個月至2年
- 需要專職IT團隊維護

**選錯平的問題：**
- 免費或低成本工具功能單一
- 數據分散，無法整合
- 缺乏本地化支援
- 不符合香港法規要求

**結果：**
- 90%的中小企選擇「乜都唔買」，繼續使用Excel和紙本
- 根據香港中華總商會的調查，超過70%的中小企表示行政工作浪費了過多人力
- 平均每間中小企每月在HR、財務管理上浪費20小時以上

### 2.2 具體痛點場景

**場景一：強積金申報**
每月底，HR需要手動計算每位員工的強積金供款，檢查是否有新入職或離職的員工，然後登入不同強積金供應商的门戶網站逐一申報。整個過程耗時2-4小時，極易出錯。

**場景二：發票跟進**
缺乏統一的發票管理系統，經常出現發出發票後客戶遲遲未付款、卻沒有人主動跟進的情況。根據調查，香港中小企的平均應收帳周轉天數高達45天，遠高於發達國家的30天標準。

**場景三：員工請假審批**
沒有系統化的請假管理，員工通過WhatsApp或電郵提交請假申請，上級需要逐一手動審批，難以追蹤剩餘年假餘額，經常出現計算錯誤。

**場景四：銷售跟進**
缺乏CRM系統，銷售人員靠筆記本或Excel記錄客戶資料，經常遺忘重要的跟進時間節點，錯失成交機會。

### 2.3 市場缺口

現有市場解決方案存在以下缺口：

| 缺口 | 市場現況 | Jikko AI解決方案 |
|------|----------|------------------|
| 價格門檻 | HK$1,000+/月 | HK$800/月起 |
| 部署時間 | 數週至數月 | 5分鐘 |
| 語言支援 | 僅英文 | 粵、普通話、英文 |
| 本地法規 | 需另購模組 | 內置HK法規 |
| 功能整合 | 需另外整合 | All-in-One |
| AI能力 | 基本或無 | GPT-4驅動 |
